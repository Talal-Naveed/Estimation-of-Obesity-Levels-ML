# Dataset Reference


## Dataset Name
**Estimation of Obesity Levels Based on Eating Habits and Physical Condition**


## Description
This dataset contains information related to the estimation of obesity levels in individuals from the countries of **Mexico, Peru, and Colombia**, based on their eating habits and physical condition.

The dataset is widely used for machine learning and data mining research related to health and lifestyle analysis.


## Dataset Details
- **Number of records:** 2,111
- **Number of attributes:** 17
- **Type:** Multiclass classification
- **Target variable:** Obesity level


## Obesity Categories
- Insufficient Weight  
- Normal Weight  
- Overweight Level I  
- Overweight Level II  
- Obesity Type I  
- Obesity Type II  
- Obesity Type III  


## Source
UCI Machine Learning Repository  
https://archive.ics.uci.edu/dataset/544/estimation+of+obesity+levels+based+on+eating+habits+and+physical+condition


## Usage in This Project
The dataset is accessed programmatically using the `ucimlrepo` Python library.  
The raw dataset files are **not included** in this repository in order to respect data source policies and keep the repository lightweight.


## Academic Use
This dataset is used as the primary data source for a university research project and accompanying machine learning experiments.
